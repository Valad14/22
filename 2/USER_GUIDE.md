# Инструкция пользователя

## 1. Просмотр карты

1. Откройте раздел **Атлас**.
2. В боковой панели выберите регион, район и карту-вопрос.
3. Включите или выключите **ареалы**, **изоглоссы**, **подписи пунктов**.
4. Наведите курсор на точку: появятся пункт, район, ландшафт, вопрос, единицы и комментарий.

## 2. Поиск карты

1. Откройте раздел **Карты**.
2. Введите код, слово из вопроса или тип вопроса.
3. Выберите карту из списка.
4. Смотрите паспорт карты и скачивайте CSV для выбранной карты.

## 3. Поиск населённого пункта

1. Откройте раздел **Пункты**.
2. Выберите регион и район.
3. Найдите пункт по названию.
4. Приложение покажет все вопросы и лингвистические единицы для пункта.

## 4. Поиск лингвистической единицы

1. Откройте раздел **Единицы**.
2. Введите форму или часть формы.
3. Выберите единицу.
4. Смотрите карту распределения и таблицу всех вхождений.

## 5. Редактирование таблицы

1. Редактируйте данные в Google Sheets.
2. Одна строка должна содержать один пункт и один вопрос.
3. После изменений нажмите **Обновить кэш данных**.
4. Раздел **Таблица** покажет ошибки обязательных полей.

## 6. Подготовка GIS-слоёв

1. Создайте или откройте слой районов/ландшафта в QGIS, ArcGIS, NextGIS или другой ГИС.
2. Проверьте координаты: EPSG:4326.
3. Экспортируйте GeoJSON.
4. Опубликуйте GeoJSON по HTTPS.
5. Вставьте ссылку в боковую панель приложения.
