# Изменённая версия приложения

Файлы из этой папки можно загрузить в папку `2` репозитория поверх текущих файлов.

## Что поменялось

- городские точки скрываются автоматически;
- демо-CSV очищен от городов и дополнен несколькими населёнными пунктами;
- оставлено по 3 вопроса для ДАРЯ и ЛАРНГ;
- номера вопросов приведены к `1`, `2`, `3` внутри каждой системы;
- видимая таблица не показывает поля `atlas_system`, `source`, `year`;
- поиск строит точки, ареалы и изоглоссы даже для короткого результата;
- добавлена форма добавления населённого пункта с поиском координат через Nominatim;
- границы регионов можно подключить через Streamlit Secret `BOUNDARY_GEOJSON_URL`; без GeoJSON строится технический контур по точкам.

## Деплой

1. Откройте GitHub → папка `2`.
2. Замените `app.py`, `streamlit_app.py`, `requirements.txt` и папку `data` файлами из архива.
3. Streamlit Cloud сам перезапустит приложение.

## Опциональные секреты Streamlit

```toml
DATA_CSV_URL = "https://.../sample_dialects.csv"
BOUNDARY_GEOJSON_URL = "https://.../regions.geojson"
NOMINATIM_USER_AGENT = "dialect-map/1.0 your-email@example.com"
```
